# BRAND_IDENTITY_V1

**Status:** Foundation document — fixes the canonical brand identity of the project.
**Scope:** Identity/brand fixation only. No architecture, no code.

---

## 1. Official project / platform name

**Python Method** is the official name of the project and platform.

This is the name to use for the project, the platform, the repository identity, and any legal/platform context.

---

## 2. Slogan

«Реабилитация без границ» (Rehabilitation Without Borders) is the **slogan** — a meaning formula expressing the spirit of the project.

It may be used as a slogan in marketing and trust contexts where appropriate.

---

## 3. Rule

**Do not use the slogan as the legal or platform name.**

«Реабилитация без границ» is a slogan, not a name. It must never be presented as the official name of the center, project, platform, or legal entity. The official name is **Python Method**.

---

## 4. Application

- **Official name (use as name):** Python Method
- **Slogan (use as slogan only):** «Реабилитация без границ» / Rehabilitation Without Borders
- Where earlier documents referred to the center *by the slogan as its name*, that is corrected to refer to **the center / platform Python Method**, with the slogan kept as a slogan where appropriate.

This document is the canonical reference for project identity.
